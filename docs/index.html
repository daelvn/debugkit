fsprint (keywords, ...)
Special case of fprint. Gets a list of keywords and won't print anything if at least one of the keywords appears in a string. Curried function.

Parameters:

  • keywords table Keyword list.
  • ... Variables to filter and print.

Table with functions to format entire strings. Shares fields with ansicolors' formatting options. So colorall.red becomes (x) -> color ("%{red}" .. x)
